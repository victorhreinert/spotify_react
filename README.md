# spotify_react
Imersao_Alura _REACT


### 💾 Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:victorhreinert/spotify_react.git
```

Entre no diretório do projeto

```bash
  cd spotify_react
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
